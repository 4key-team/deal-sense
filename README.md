# Deal Sense

Инструмент для команд, которые регулярно участвуют в тендерах и готовят коммерческие предложения.

**Версия:** 0.8.2

## Проблема

Подготовка заявки на тендер — это часы рутины: прочитать 40 страниц ТЗ, понять подходит ли команда, оценить риски, заполнить шаблон КП. Малые и средние команды (агентства, интеграторы, продакшены) тратят на это 1-2 дня, а потом получают отказ, потому что не заметили требование по SOC 2 на 37-й странице.

## Решение

Deal Sense делает две вещи:

**Анализ тендера** — загружаете PDF/DOCX документацию, указываете профиль компании. Получаете за минуту:
- Вердикт GO / NO-GO с обоснованием
- Список преимуществ и рисков
- Матрицу соответствия требованиям (met / partial / missing)
- Оценку трудозатрат на подготовку заявки

**Генератор КП** — загружаете .docx-шаблон + контекст (кейсы, прайс, бриф клиента). Получаете заполненный документ с цветовой разметкой: что взято из шаблона, что сгенерировано AI, что требует ревью.

## Почему это должно существовать

- **Экономия времени** — анализ тендера за 1 минуту вместо 4 часов
- **Меньше ошибок** — AI не пропустит требование на 37-й странице
- **Локально-первый** — API-ключ хранится в браузере, данные не уходят на чужой сервер (кроме выбранного LLM-провайдера)
- **Без привязки к вендору** — работает с Anthropic, OpenAI, Google Gemini, Groq, Ollama (локально)

| DOCX чтение | stdlib zip + xml |

## API

| Метод | Путь | Описание |
|-------|------|----------|
| POST | /api/tender/analyze | Анализ тендера (multipart: files + company_profile) |
| POST | /api/proposal/generate | Генерация КП (multipart: template + params JSON) |
| POST | /api/llm/check | Проверка подключения к LLM |
| GET | /api/llm/providers | Список доступных провайдеров |

## Быстрый старт

```bash
# Клонировать
git clone <repo-url> && cd deal-sense

# Настроить LLM
cp .env.example .env
# отредактировать .env — указать API-ключ

# Backend
just dev-backend    # :8080

# Frontend
just dev-frontend   # :5173
```

## Команды

```bash
just dev              # Backend + Frontend параллельно
just test             # Backend тесты (164 теста)
just test-frontend    # Frontend тесты (153 теста)
just test-all         # Все тесты
just test-cover       # Backend покрытие
just build            # Сборка
just lint             # go vet
just typecheck        # tsc --noEmit
just docker-up        # Docker Compose
```

## Конфигурация

Через переменные окружения (см. `.env.example`):

| Переменная | По умолчанию | Описание |
|------------|-------------|----------|
| PORT | 8080 | Порт сервера |
| LLM_PROVIDER | anthropic | Провайдер: anthropic, openai, gemini, groq, ollama, custom |
| LLM_BASE_URL | — | URL API провайдера |
| LLM_API_KEY | — | API-ключ |
| LLM_MODEL | claude-sonnet-4-5 | Модель |
| LOG_LEVEL | info | Уровень логирования: debug, info, warn, error |
